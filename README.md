# Lestari Admin

Animal conservation mobile application.

## Project Requirements
- Flutter 3.16.3
- Dart 3.2.3


## Setup Guide
- Clone project ini.
- Buka di Android Studio atau Visual Studio Code.
- Jalankan `flutter pub get`.
- Lalu run aplikasinya.
